Search in VR

UX Exploration

Today, text input in VR is a challenging task. This exploration aims to test out new UX concepts to improve text input.

The Problem

Text input is important in a number of scenarios such as searching for items, inputting pins, etc. When we start up our VR experience, one of our first tasks is to find the app we want to launch either by browsing or searching. I was inspired to start this project when I noticed how much I struggled to search for apps in VR, and how unnatural it felt to input text. This awkwardness felt similar to text input for gaming consoles (e.g. Xbox) where typically you use an analog stick or d-pad to select characters.

How does text input work in VR? We use keyboards to input text on desktop and on mobile, so most VR apps provide a giant digital QWERTY keyboard to input text. Users will select characters either by

  1. Using our wrist or arms to move a laser pointer to select
  2. Push an analog stick to select (similar to the console solution)
  3. Hover over a character and clicking a touchpad to select
