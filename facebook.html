Facebook Sentiments

Data Science

Data Scientist Intern at Visual Revenue. Provided and presented internal report on Facebook API details and findings of Facebook Post content for predicting Likes.

News Properties share news articles to their subscribers across many digital media (ex. web and social). Can we help News Properties understand which articles will drive the most Facebook Likes, a metric that represents the popularity of a post? Can we use only the content to predict Likes?

I ramped up the development team on Facebook API by reporting on API nuances and collecting data on US News Properties. I also developed a model to investigate the effectiveness of predicting Facebook Likes based on content and other Post features using sentiment analysis and regression analysis
